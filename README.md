### DFWP 
Wordpress theme framework

#### NPM INSTALL
npm install --save-dev

#### BUILD
Se mettre dans le dossier build

	cd /build
	
Commandes disponibles :

	grunt phpdocumentor (générer la doc)

----------

#### DOCUMENTATION
A lire ici après l'avoir générer
\docs\index.html
